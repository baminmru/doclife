###1 Главное окно
Сейчас Вы можете воспользоваться клавишами переключения окон
            Alt + ->  или  Alt + <-
либо вызвать список активных окон при помощи Alt + .
  Выход из программы возможен при нажатии Alt + X
###2 Главное Меню
    Вы находитесь в главном меню программы DocLife!
Главное меню служит менеджером запуска окон программы.
При помощи стрелок  и  Вы можете перемешаться по пунктам
меню. Нажатием Enter Вы выбираете один из пунктов меню и
таким образом либо попадаете в меню второго уровня либо
вызываете на экран соответствующее окно.
Вы можете также, воспользоваться клавишами переключения
окон:         Alt + ->  или  Alt + <-
либо вызвать список активных окон при помощи Alt + .
  Выход из программы возможен при нажатии Alt + X

      Лаборатория Макроэкономических Исследований
    желает Вам  приятной работы с нашей программой!
###3 Меню Договора
Из этого подменю Вы можете запустить основные окна программы:
Ведение договоров - окно где фиксируется состояние договоров
Закрытые договора - окно для работы с завершенными договорами
Счета - окно где фиксируются разовые счета
Отчет о задолженностях - формируется отчет о задолженностях,
План по договорам  - формируется отчет о плане затрат
Формировать Предложения к оплате - программа формирует список
  сумм которые можно оплатить из имеющихся средств.
Работа с Предложением к оплате - Вы можете редактировать и
  изменять список сумм, сформированный ранее.
###4 Меню Смета
Вы можете выбрать пункт Смета, для внесения дополнений,
создания новой сметы расходов или получения справки по
текущему состоянию сметы.
Либо пункт Исполнение сметы, для получения справки об
исполнении сметы расходов.
###5 Меню приоритеты
Вы можете выбрать один из двух пунктов:
Приоритеты по показателям - задание приоритетов
по признакам договоров для формирования предложений
на оплату, либо
Тип финансирования - задание способов оплаты
которые могут быть использованы при оплате договоров.

###19
Вы находитесь в окне редактирования.
В программе имеется несколько типов полей ввода
1. Простое поле куда можно вводить текстовую или числовую информацию.
2. Логическое поле (Да/Нет), здесь необходимо использовать пробел
для изменения значения поля.
3. Поле выбора (курсор в нем не видим), здесь следует использовать
пробел для изменения значения поля, либо Alt +  для просмотра
и выбора из списка возможных вариантов. Выбор из списка производится
при нажатии Enter, Esc - сохраняет старое значение поля.
4. Комбинированное поле работы со справочниками, здесь разрешен
ввод кодов, при этом программа проверяет наличие кода в справочнике,
либо переход к работе со списком при нажатии Alt + , работая со
справочником вы можете использовать F5 для поиска.
- Переход между полями осуществляется при нажатии клавиш:
      Tab   Shift + Tab
- Очистить поле можно нажатием Shift + Delete
- Работа в окне редактирования заканчивается при нажатии клавиш:
Enter - при этом результаты ввода учитываются программой или
Esc   - при этом программа игнорирует введенную информацию

###21 Выборка
Вы находитесь в меню задания условий выборки.
Вы можете выбирать поля нажатием Enter для просмотра или
установки условия по этому полю.
Поля участвующие в выборке помечены значком √.
Нажатием Esc вы завершаете процесс задания условия
и переходите либо в режим создания документа по выборке
либо возвращаетесь в окно работы с договорами
###22 Условие для поля выборки
Вы находитесь в окне задания условия для конкретного поля.
Если вы хотите чтобы это условие участвовало в выборке,
1. установите 'Да' напротив строки 'Использовать для выбора'
2. задайте конкретное условие для поля, если требуется задать
промежуток значений проследите чтобы условие могло выполняться
хотя бы для одного значения иначе результат выборки будет нулевым.
3. после задания условия нажмите Enter или Esc

###34 Регистрация пользователя
Вы регистрируете нового пользователя
Присвойте ему уникальный идентификационный код,
Определите (или не определяйте) его пароль,
И обязательно задайте уровни доступа на все файлы базы данных.
(Справа от названия файла отображается вариант доступа к файлу.)
После определения уровней доступа нажмите Enter чтобы
зафиксировать информацию в базе данных.
###35
Вы можете изменить уровни достпа пользователя
для этого выбирете название нужного вам файла и установите для
этого файла нужный уровень доступа.
(Справа от названия файла отображается вариант доступа к файлу.)
Проделайте операцию для всех названий файлов которые пользователю
можно (или нельзя) обрабатывать.
После определения уровней доступа нажмите Enter чтобы
зафиксировать информацию в базе данных.
